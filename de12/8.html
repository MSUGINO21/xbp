※ソースコードの参考：
  • 斎藤さんの「デザイン演習Ⅰ・Ⅱ」のHPから。リンクはこちら（外部サイト）
  • 元データを１つの路線のワンルームに絞ってから、駅ごとに平均の棒グラフを出すコード↓
    （篠原さんから共有：例 家賃）
  1. import pandas as pd # データ分析に用いるライブラリ,padnasをpdと認識する
  2. import matplotlib.pyplot as plt # グラフ表示に用いるライブラリ
  3. pd.set_option('display.unicode.east_asian_width', True) # 表示のずれを少し緩和
  4. plt.rcParams['font.family'] = 'IPAexGothic' # グラフ表示におけるフォントの指定
  6. data_path = "./data.csv"
  7. df_data = pd.read_csv(data_path, encoding="utf-8-sig")
  9. keikyu = (df_data["路線"] == "京急本線") & (df_data["間取り"] =="ワンルーム")
  10. df_keikyu = df_data[keikyu]
  11. df_keikyu_mean = df_keikyu.groupby("駅").mean()
  12. df_keikyu_mean.loc[:, "合計時間"].sort_values().plot.bar()
  13. plt.subplots_adjust(bottom=0.3)
  14. plt.show()
