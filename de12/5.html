１班で作成したゲーム：なぞなぞゲーム

概要

なぞなぞクイズの回答を入力し、その正誤を判断してくれるゲームを作成した。（全４問）
最後に正解数と正答率を教えてくれる。

ソースコード↓
  1. ok = 0
  2. ng = 0
  4. print("なぞなぞゲーム")
  5. a = input("赤ちゃんしかいない国はどこ？")
  6. b = input("クマが問題を解けない場所は？")
  7. c = input("全員が女の子の四つ子がお正月に披露するものは何でしょうか？")
  8. d = input("パパとママは２回ずつ会えるのに、父と母は１回も会えませんこれ何のこと？")
  10. if a == "ニュージーランド":
  11.     print("正解")
  12.     ok = ok + 1
  13. else:
  14.     print("不正解")
  15.     ng = ng + 1
  18. if b == "熊本県" or b == "熊本" or b == "くまもとけん" or b == "くまもと":
  19.     print("正解")
  20.     ok = ok + 1
  21. else:
  22.     print("不正解")
  23.     ng = ng + 1
  26. if c == "獅子舞" or c == "ししまい" in c:
  27.     print("正解")
  28.     ok = ok + 1
  29. else:
  30.     print("不正解")
  31.     ng = ng + 1
  34. if d == "唇" or d == "くちびる" or d == "口":
  35.     print("正解")
  36.     ok = ok + 1
  37. else:
  38.     print("不正解")
  39.     ng = ng + 1
  41. print("おわりーーー　あなたの正解数は", ok, "正解率は", ok/(ok+ng)*100, "％です")

補足
  • okとngの変数を用いて、毎回回答するごとに点数が加算されるようにした。
  • 複数の表記が想定される回答は、if文でorを用いて複数の記載に対応できるようにした。
  • 掛け算は＊、割り算は/で示す。
