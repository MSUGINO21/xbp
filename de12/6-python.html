python メモ

pyファイルをデスクトップアプリにするには

  1. pyinstallerをインストールする。
    (Mac)ターミナルを開き、pip install pyinstallerと入力→Enterでインストールする。
  2. ターミナルでpyファイルがあるディレクトリへ移動する。
    例）（ホームディレクトリのPycharmProjectsのtodoファイル→cd PycharmProjects/todoとターミナルに入力→Enter）
  3. ターミナルに、pyinstaller pyファイル名 --onefile --noconsoleと入力→Enter
  4. 該当フォルダのdistフォルダにアプリケーションファイルが完成！
