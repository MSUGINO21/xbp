リンクのタグ

リンク挿入のタグは、
<a href="URL">リンクとして表示したい文字</a>
です。

例１）Yahoo!Japan（https://www.yahoo.co.jp/）のトップページを表示する場合は
以下のように入力する。 <a href="https://www.yahoo.co.jp/">yahooトップページ</a> ＜実際の見え方＞
yahooトップページ


例２）　imagesフォルダ内の画像をYahoo!Japan（https://www.yahoo.co.jp/）のトップページのリンクに
する場合は以下のように入力する。 <a href="https://www.yahoo.co.jp/"><img src="images/dolphin.png" width="30%"></a> ＜実際の見え方＞   ※下の画像をクリックするとYahooのサイトへ移動




<a href="URL">リンクとして表示したい文字</a>では、リンク先は同じウインドウで開かれますが、
<a href="URL" target="_blank">リンクとして表示したい文字</a>だと、別ウインドウで開かれます。

例２）imagesフォルダ内の画像をYahoo!Japan（https://www.yahoo.co.jp/）のトップページのリンクに
する場合は以下のように入力する。 <a href="https://www.yahoo.co.jp/" target="_blank">yahooトップページ</a> ＜実際の見え方＞   ※下の画像をクリックするとYahooのサイトへ移動

yahooトップページ
